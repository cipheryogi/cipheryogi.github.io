Xpediter — Merging User Profiles

Category: Xpediter Debugging

Merging Other User's Profiles

  1. Access the Utility Menu: enter option 5 on the primary menu
  2. Select Merge Option: choose option 7 "Merge profiles from alternate users"
  3. Specify Dataset: enter the dataset name in format userid.tsolib.ispfprof
  4. Review and Select: a screen displays all profiles, allowing you to select one or multiple to merge

Using Test Scripts

Test scripts eliminate the need to repeatedly perform debugging actions (forcing data into variables, setting breakpoints, creating interrupts) each time you enter XPEDITER.

Setup Process

  1. Include your test script library in the SETUP MENU's INCLUDE option
  2. Run your program in XPEDITER and validate program logic
  3. Upon exiting, a screen prompts about creating a log and script
  4. Modify the generated script to match your requirements
  5. Save in your TEST SCRIPT library
  6. For subsequent runs, type INCLUDE MEMBERNAME in the command line
