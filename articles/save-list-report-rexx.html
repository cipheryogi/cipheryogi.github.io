Generating SAVE LIST Report Using REXX

Category: REXX Scripting

Overview

A REXX script that creates a dataset attributes report comparable to ISPF's "SAVE LIST" command functionality.

Complete Script

/* REXX MAIN START                                            */
/* THIS REXX GENERATES A REPORT OF DATASET ATTRIBUTES EXACTLY */
/* SIMILAR TO "SAVE LIST" COMMAND (ISPF)                      */
/* THE REPORT IS GENERATED IN DATASET                         */
/* <USERID.GROUPNAME.MEMBERS>                                 */
/* E.G. GROUPNAME IS "YOGESH" BELOW                           */
/*************************************************************/

SAY 'PLEASE SPECIFY THE PDS:'
PULL INPDSN

WSMEM = ' '

ADDRESS ISPEXEC
"LMINIT DATAID(INID) DATASET('"INPDSN"') ENQ(SHRW)"
"LMOPEN DATAID("INID") OPTION(INPUT)"
"LMMLIST DATAID("INID") MEMBER(WSMEM) OPTION(LIST)"
"LMMLIST DATAID("INID") MEMBER(WSMEM) OPTION(FREE)"
"LMMLIST DATAID("INID") MEMBER(WSMEM) OPTION(SAVE) GROUP(YOGESH)"
"LMCLOSE DATAID("INID")"
"LMFREE DATAID("INID")"

RETURN

Key Components

  • Prompts user to specify the PDS
  • LMINIT initializes data access with shared read-write enqueuing
  • LMOPEN opens the dataset for input
  • LMMLIST retrieves member listings with multiple options
  • Results stored in <USERID.GROUPNAME.MEMBERS>
