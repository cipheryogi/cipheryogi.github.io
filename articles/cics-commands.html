CICS Commands Reference

Category: CICS

Overview

A reference guide to the common CICS (Customer Information Control System) commands and their COBOL syntax.

Notation Conventions

  • [ ] — indicate optional parameters
  • [ | ] — indicates a choice of one of the options listed (or none)
  • { | } — indicates you must specify one of the options

If you are using PL/1, substitute ; for END-EXEC.

Common CICS Commands

This reference covers the most frequently used CICS commands in COBOL CICS application programming, organized by their function in transaction processing, BMS map handling, file control, temporary storage, and program control operations.
