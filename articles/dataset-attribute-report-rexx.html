Dataset Attribute Report Using REXX in Batch

Category: REXX Scripting

Overview

Execute a REXX program in batch mode via JCL to generate reports of dataset attributes using the LISTDSI function.

JCL Structure

  • Execution Program: IKJEFT1A (TSO batch interface)
  • Input Dataset: Contains list of dataset names (LRECL=22)
  • Output Dataset: Generated report (LRECL=150)
  • REXX Library: Allocated via SYSEXEC

Report Columns

ColumnDescription
Dataset NameFull qualified name
Creation DateMM/DD/YY format (converted from Julian)
LRECLLogical record length
Block SizePhysical block size
Space UnitAllocation unit type
PrimaryPrimary space allocation
SecondarySecondary space allocation
ExtentsNumber of extents used
UsedUsed space
MembersMember count (for PDS)

Key Technical Details

  • Uses LISTDSI with DIRECTORY parameter for member counting
  • Julian dates converted to MM/DD/YY format
  • Dataset allocation uses "MOD REUSE" to append records

TSO Quick Registration

TSO ALLOC FI(SYSEXEC) DS('VNDJ29.REXX.UTILS') SHR REU
