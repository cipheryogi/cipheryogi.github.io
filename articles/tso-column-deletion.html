TSO Command — Column Deletion

Category: JCL and z/OS Utilities

The Command

C ALL P'=' '' 35 80

Syntax Breakdown

  • C ALL = Change all occurrences
  • P'=' = Find pattern (equals sign)
  • '' = Replace with nothing (empty string)
  • 35 80 = Column range (columns 35 through 80)

Example

Given a file with records like:

B0BP030B               01.01 05/07/22 03:59    25    25     0 VNDJ29
D0AZU520               01.60 06/02/01 02:57    57   160     0 VNDJ29
D0AZU530               01.54 06/02/01 02:58    58   160     0 VNDJ29

The command isolates job names (appearing in the first 8 columns) by eliminating data in columns 35-80.
