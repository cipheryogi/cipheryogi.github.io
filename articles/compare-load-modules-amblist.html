Compare Load Modules — AMBLIST Utility (z/OS)

Category: JCL and z/OS Utilities

Overview

Using the z/OS AMBLIST utility for analyzing and comparing load modules stored in a partitioned dataset.

Sample JCL

//VNDJ29S   JOB (DIELD),'REPORT JOB',
//    MSGCLASS=T,CLASS=S,NOTIFY=&SYSUID
//
//STEP010  EXEC PGM=AMBLIST
//DD1      DD DISP=SHR,DSN=VNDJ29.LOADLIB
//SYSPRINT DD DISP=(NEW,CATLG,DELETE),UNIT=SYSDA,
//            SPACE=(CYL,(5,5),RLSE),
//            DCB=(DSORG=PS,RECFM=FBA,LRECL=121,BLKSIZE=32670),
//            DSN=VNDJ29.COBOL.REPORT
//SYSIN    DD *
  LISTIDR OUTPUT=ALL,DDN=DD1

AMBLIST Capabilities

  • Formatted listing of object modules
  • Control section (CSECT) mapping within load modules or program objects
  • Module inventory within link pack areas (LPA)
  • DAT-on nucleus content mapping

The LISTIDR OUTPUT=ALL,DDN=DD1 command generates a comprehensive report on all components within the specified load library.
