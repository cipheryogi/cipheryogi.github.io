Expanding PROC in JCL Using Xpediter

Category: Xpediter Debugging

Step-by-Step Instructions

  1. Launch XPEDITER and navigate to Option 2 (TSO)
  2. Specify the JCL file — In the "File List/JCL Member" field, enter the path to your JCL, e.g.: 'P4M0.CP000000.P4M2.JCL(P4MA1201)'
  3. Enter a program name under the PROGRAM heading — actually, any program name would do
  4. Wait for allocation — Press Enter, you'll receive "Allocating User Datasets". Duration depends on the number of files in the PROC
  5. Access the expanded JCL — Type EDIT on the command line and press Enter
  6. Finalize — The JCL now appears in expanded form. Delete the PROC statement if still present

Use Cases

The expanded JCL format is useful for XPEDITING batch programs or testing purposes.
