XML Script to Create a ChangeMan ZMF Package

Category: Serena ChangeMan

Overview

A complete XML template for creating a Serena ChangeMan package using batch XML services.

XML Structure

<?xml version="1.0"?>
<service name="PACKAGE">
 <scope name="SERVICE">
  <message name="CREATE">
   <header>
    <subsys>J</subsys>
    <test> </test>
    <product>CMN</product>
   </header>
   <request>
    <applName>XXX1</applName>
    <createMethod>2</createMethod>
    <packageModel>XXX1000004</packageModel>
    <packageLevel>1</packageLevel>
    <packageType>1</packageType>
    <packageTitle>TEST PACKAGE USING SERXMLBC</packageTitle>
    <packageDesc>TESTING XML SERVICES</packageDesc>
   </request>
  </message>
 </scope>
</service>

Package Configuration Options

FieldOptions
Package Level1 (Simple), 2 (Complex), 3 (Super), 4 (Participating)
Package TypePlanned or Unplanned
Time SpanPermanent or Temporary
SchedulerCMN, Manual, or Other
Create Method1 (Short), 2 (Long — requires all tags)

Site Installation Section

The <siteInfo> section specifies where and when the package should be installed, including site name, install date, time window, and contact information.

Scheduling

The <schedulingInfo> section defines predecessor and successor job names for workflow integration.
