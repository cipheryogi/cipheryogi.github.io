Compressing CICS Libraries (PDS)

Category: CICS

Overview

How to compress Partitioned Data Sets (PDS) libraries used by CICS regions on z/OS systems to reclaim unused space.

Compress Job

//*****************************************************
//* COMPRESS CICS LIBRARIES
//*****************************************************
//JOBNAME  JOB (XXXX),'COMPRESS DATASET',
//    MSGCLASS=X,CLASS=X,NOTIFY=&SYSUID
//*************************************************************
//COPYSTEP EXEC PGM=IEBCOPY,PARM='SIZE=9999K'
//SYSPRINT DD   SYSOUT=T
//DISK1   DD   DSN=Library Name,
//             DISP=SHR
//SYSUT3   DD  UNIT=SYSDA,SPACE=(CYL,(5,1))
//SYSUT4   DD  UNIT=SYSDA,SPACE=(CYL,(5,1))
//NFDRCOPY DD DUMMY
//SYSIN    DD   *
  C O=DISK1,I=((DISK1,R))
/*

Key Components

  • Program: IEBCOPY utility with SIZE=9999K parameter
  • Input Dataset: The library to be compressed (DISK1)
  • Disposition: SHR (shared) access allows compression of active libraries
  • Temporary Datasets: SYSUT3 and SYSUT4 allocated on SYSDA
  • Command: C O=DISK1,I=((DISK1,R)) copies and compresses the dataset
