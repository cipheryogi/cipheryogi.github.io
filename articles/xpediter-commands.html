Xpediter Commands Reference

Category: Xpediter Debugging

Breakpoint Commands

  • AFTER — Establishes breakpoints following statement execution
  • BEFORE — Establishes breakpoints preceding statement execution
  • WHEN — Sets conditions that pause execution at specified points

Execution Control

  • GO (PF12) — Resumes execution. GO 5 1 executes five statements with one-second intervals. GO UNTIL WA-HOURS executes until a variable changes
  • GOTO — Repositions execution pointer to a different statement
  • SKIP — Temporarily bypasses a specific statement

Data and Analysis

  • CALC — Performs hex and decimal calculations
  • COUNT — Establishes execution counts and gathers test coverage statistics
  • KEEP — Views data items in a scrollable, sizeable keep window with modification capability
  • USING — Maps data according to record formats from COBOL programs

Display and Navigation

  • EXCLUDE — Hides specified lines. Excluded lines reappear during execution
  • FIND — Positions cursor on specified strings
  • LOCATE * (PF6) — Transfers to Source Listing screen
  • SHOW — Modifies screen formatting for different data views
  • SET — Adjusts debugging parameters

Session Management

  • EXIT (PF4) — Transfers to Exit Session screen
  • HELP (PF1) — Displays hypertext online help
  • XPND — Terminates the XPEDITER/CICS session
  • NEWC — Loads a new program version, restores breakpoints. Format: NEWC program-name
  • DMAP — Displays a BMS map
