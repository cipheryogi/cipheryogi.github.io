Compressing Libraries (Prod, QA, Dev)

Category: JCL and z/OS Utilities

Overview

JCL job for compressing baseline and production libraries associated with a ChangeMan application using the IEBCOPY utility.

Sample JCL

//*****************************************************
//* COMPRESS ALL BASELINE AND PRODUCTION LIBRARIES
//*****************************************************
//STEP010 EXEC PGM=IEBCOPY
//SYSPRINT DD  SYSOUT=*
//IN1      DD  DISP=SHR,DSN=PR00.CP000000.PISB.SRC
//OUT1     DD  DISP=SHR,DSN=PR00.CP000000.PISB.SRC
//SYSIN    DD  *
 COPY INDD=((IN1,R)),OUTDD=OUT1
/*
//STEP011 EXEC PGM=IEBCOPY
//SYSPRINT DD  SYSOUT=*
//IN1      DD  DISP=SHR,DSN=PRCM.CP000000.PISB.SRC.DELTA
//OUT1     DD  DISP=SHR,DSN=PRCM.CP000000.PISB.SRC.DELTA
//SYSIN    DD  *
 COPY INDD=((IN1,R)),OUTDD=OUT1
/*

Key Points

  • Both steps use IEBCOPY for PDS compression
  • The R parameter indicates read-only mode during copy
  • Naming convention covers production (PR00, PRCM) libraries
  • Handles both source code libraries (.SRC) and delta/change tracking (.DELTA)
