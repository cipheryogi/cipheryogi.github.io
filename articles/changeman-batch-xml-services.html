Using Serena-ChangeMan Batch XML Services

Category: Serena ChangeMan

Overview

How to use the SERXMLBC batch program with XML scripts to automate package creation in Serena ChangeMan v5.6.0+. Restricted to ChangeMan administrators with appropriate access authority.

Sample JCL

//SERXMLX   JOB (xxxxx),'SERXMLBC',
//    MSGCLASS=T,CLASS=S,NOTIFY=&SYSUID
//STEPXML EXEC PGM=SERXMLBC
//STEPLIB  DD DISP=SHR,DSN=SYS3.CHGMAN.xxxx.V560.LINKLIBC
//         DD DISP=SHR,DSN=SYS3.CHGMAN.xxxx.V560.LINKLIB
//         DD DISP=SHR,DSN=SYS3.CHGMAN.xxxx.V560.SERCOM.LINKLIBC
//         DD DISP=SHR,DSN=SYS3.CHGMAN.xxxx.V560.SERCOM.LINKLIB
//SER#PARM DD DISP=SHR,DSN=SYS4.CHGMAN.XXXX.TCPIPORT
//SERRPLY  DD SYSOUT=*,DCB=(RECFM=VB,LRECL=4080,BLKSIZE=4084)
//SYSPRINT DD SYSOUT=*
//SERPRINT DD SYSOUT=*
//SYSUDUMP DD SYSOUT=*
//XMLIN    DD DISP=SHR,DSN=SERXML.XMLIN(PKGCREAT)
//XMLOUT   DD DISP=SHR,DSN=SERXML.XMLOUT

Successful Output

<?xml version="1.0"?>
<service name="PACKAGE">
  <scope name="SERVICE">
    <message name="CREATE">
      <result>
        <package>XXX1000006</package>
        <applName>ITS1</applName>
        <packageId>000006</packageId>
      </result>
      <response>
        <statusMessage>CMN2100I - XXX1000006 change package created</statusMessage>
        <statusReturnCode>00</statusReturnCode>
      </response>
    </message>
  </scope>
</service>

Key XML Tags

TagPurpose
<requestorDept>Enter the RLSE value
<workChangeRequest>GSD change ticket value
<packageDesc>Description (1-46 lines)
<packageImplInst>Installation instructions
<createMethod>1=Short, 2=Long (requires all tags)
