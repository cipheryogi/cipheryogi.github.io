Setting Up XMLSERV — ChangeMan XML Services

Category: Serena ChangeMan

Overview

Configuration guide for customizing the XMLSERV REXX member to enable online XML services. Restricted to system administrators.

Configuration Variables

VariablePurposeExample
mapdsXML mapping data datasetPoints to .MAPDATA dataset
serparmChangeMan STC parameter datasetEnds with TCPIPORT
xmlindsUser XML input datasetDefault: XMLIN
xmloutdsUser XML output datasetDefault: XMLOUT
subsysChangeMan subsystem IDSingle byte identifier
sysoutOutput classDefault: H

Library Configuration

  • ALTLIB: Reference the CLIST library containing XMLSERV and concatenate the CEXEC library
  • ISPLLIB: Include all vendor load libraries plus SERCOM linklibs
  • ISPMLIB: Provide vendor message libraries (ending with ISPMLIB node)
  • ISPPLIB & ISPSLIB: Include SERCOM panel libraries and vendor SERCOM.ISPSLIB (critical)

Execution

Launch using either:

  • Line command: EX
  • TSO command: TSO EXEC '<PDS>(XMLSERV)'
