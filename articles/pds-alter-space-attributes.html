PDS Command — Alter Space Attributes

Category: JCL and z/OS Utilities

Overview

Using PDS commands to modify the space attributes of datasets on mainframe systems.

The PDS command utility allows administrators to view and alter space allocation attributes for partitioned datasets, including primary and secondary space, directory blocks, and other DSCB fields.

Practical demonstrations of these commands were originally provided via attached documentation showing step-by-step usage on a live system.
