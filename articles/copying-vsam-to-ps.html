Copying VSAM Dataset into PS Dataset

Category: JCL and z/OS Utilities

Overview

How to convert VSAM indexed files into PS (Physical Sequential) flat files using FILE-AID, particularly when standard IDCAMS REPRO fails.

Procedure

  1. Use FILE-AID Copy Function — Access the regular file copy option in FILE-AID using selection 3.3 on the indexed file
  2. Configure Output Dataset — Specify the output dataset name with DISPOSITION=OLD, select 'B' (BATCH) mode, and press Enter
  3. Handle Error Message — You will receive: "ER852-OUTPUT DATASET NOT AVAILABLE, TAPE ASSUMED." This can be safely ignored. Type JCL on the command line and press Enter
  4. Modify JCL Parameters:
    • Remove the output dataset's DISP, UNIT, and LABEL parameters
    • Add (NEW,CATLG,DELETE) disposition
    • Include required SPACE and DCB parameters
